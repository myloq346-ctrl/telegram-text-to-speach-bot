# Telegram Text To Speech Bot

A Telegram bot that converts Persian and English text into speech using gTTS and Edge-TTS.

## Features

- Convert text to speech
- Persian voices
- English voices
- MP3 output
- Fast response

## Tech Stack

- Python
- pyTelegramBotAPI
- Edge-TTS
- gTTS
- python-dotenv

## Installation

pip install -r requirements.txt

## Usage

python core/main.py
